# WSO Yappinator

Get a variety of characters to yap at you while you fly!

* [Download mod](https://github.com/nikkorap/NO-WSO-Yappinator/releases/latest)
* [Browse Voicepacks](VoicePacks) (to download, click on the desired pack, then click "download raw file" on the right)

place the folders found in audiopacks inside a folder named "audio" next to the mod .dll
Folder structure should look like this:
```
bepinex/plugins/
  └─ WSOYappinator/
    ├─ WSOYappinator_version.dll
    └─ audio/
       ├─ Prez/
       │  ├─ eventPriorities.txt
       │  └─ audiofiles.mp3
       └─ Galaxy/
          ├─ eventPriorities.txt
          └─ audiofiles.wav
```
